# ⚡ COMECE AGORA - 3 Passos Rápidos

## Windows Users 🪟

### Passo 1: Duplo clique em `iniciar.bat`
A mágica acontece automaticamente! ✨

### Passo 2: Aguarde a primeira execução
Pode levar 30-60 segundos para instalar dependências

### Passo 3: Abra seu navegador
```
👤 Usuário:    http://localhost:3000/usuario
👨‍💼 Atendente:  http://localhost:3000/atendente
📺 Painel:     http://localhost:3000/painel
⚙️ Admin:      http://localhost:3000/admin
```

---

## Mac/Linux Users 🍎🐧

### Passo 1: Abra Terminal

### Passo 2: Na pasta do projeto, execute:
```bash
npm install
npm start
```

### Passo 3: Acesse as mesmas URLs acima

---

## 🎮 Testando Agora

### Abra 3 abas:
1. **Aba 1** - Usuário: Retire uma senha
2. **Aba 2** - Atendente: Faça login e chame a senha
3. **Aba 3** - Painel: Veja a senha aparecer em tempo real

### Pronto! 🎉
Seu sistema de fila está funcionando!

---

## 🔑 Senha Admin

Para resetar senhas (no painel admin):
```
AGR.Senhas
```
